# 🧠 AI-Based Memory Usage Forecaster

A machine learning–powered system that predicts future memory usage and enables **proactive memory management** instead of reactive handling.

---

## 🚀 Features
- 📊 Collects real-time system memory data  
- ⚙️ Generates time-series features  
- 🤖 Uses **Random Forest & LSTM** for prediction  
- 🧠 Decision engine suggests actions (swap, pre-allocate, etc.)  
- 📈 Model evaluation with performance metrics  

---

## 🛠️ Tech Stack
- Python  
- psutil, pandas, numpy  
- scikit-learn, PyTorch  
- matplotlib
